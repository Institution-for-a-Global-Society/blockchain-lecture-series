- インターフェースに定義された関数を実装する

### 2. `ERC20`: ERC20に触れる

- 関数 `approve` と `transferFrom` を実装してみる

## コマンド一覧

| コマンド | 説明                                  |
| -----------: | :---------------------------------------- |
|  `yarn dev` | ローカルネットワークの立ち上げ |
|  `yarn compile` | コントラクトのコンパイル |
|  `yarn deploy:token` | `Token` コントラクトのデプロイ |
|  `yarn deploy:erc20` | `ERC20` コントラクトのデプロイ |
|  `yarn run:token` | `Token` コントラクトに対するスクリプト実行（挙動確認用） |
|  `yarn run:erc20` | `ERC20` コントラクトに対するスクリプト実行（挙動確認用） |
