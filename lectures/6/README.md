# Lecture 6~7: RWA

## コーディング演習: `VPP`コントラクト

1. [`VPP`コントラクト](./contracts/VPP.sol)を読み込みましょう
2. 売買されている構造体 `Listing` を、ERC-20 トークンとして再実装してみましょう
3. 追加したい機能を1つ考案して、実装してみましょう

## コマンド一覧

| コマンド | 説明                                  |
| -----------: | :---------------------------------------- |
|  `yarn` | 必要ライブラリのインストール |
|  `yarn dev` | ローカルネットワークの立ち上げ |
|  `yarn compile` | コントラクトのコンパイル |
|  `yarn deploy:vpp` | `VPP` コントラクトのデプロイ |
|  `yarn run:vpp` | `VPP` コントラクトに対するスクリプト実行（挙動確認用） |
