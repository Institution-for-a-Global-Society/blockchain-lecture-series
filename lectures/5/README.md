# Lecture 5: マーケット

## コーディング演習: AMMコントラクト

- インターフェース `IAMM` に沿って `AMM` の各関数を実装する

## コマンド一覧

| コマンド | 説明                                  |
| -----------: | :---------------------------------------- |
|  `yarn dev` | ローカルネットワークの立ち上げ |
|  `yarn compile` | コントラクトのコンパイル |
|  `yarn deploy:amm` | `AMM` コントラクトのデプロイ |
|  `yarn run:amm` | `AMM` コントラクトに対するスクリプト実行（挙動確認用） |
